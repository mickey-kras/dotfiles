# Source Freshness

When citing sources, note their publication or last-updated date. Apply these guidelines:

- sources older than 2 years: flag as potentially outdated, verify current relevance
- sources older than 5 years: use only for historical context, not as current evidence
- rapidly evolving domains (AI, social media, crypto): prefer sources from the last 6 months
- stable domains (regulation, academic theory): older sources are acceptable if foundational

When a finding relies on dated sources, explicitly state this and note whether more recent data could change the conclusion.

Do not present stale data as current without disclosure.
