# Editorial Review Checklist

Use this skill before publishing or handing off content for final review.

Checklist:
- accuracy: all claims are sourced or flagged for verification
- brand voice: tone matches the brand guidelines or stated voice
- audience fit: language level and assumptions match the target reader
- SEO basics: title tag, meta description, heading hierarchy, alt text
- platform format: length, structure, and conventions match the target channel
- accessibility: reading level, image descriptions, link text clarity
- legal: no unverified claims, proper attribution, no copyrighted material
- call to action: clear next step for the reader

Process:
1. read the content end to end
2. check each item above and note pass or concern
3. produce a summary with green (pass), yellow (minor), red (blocking) items
4. if any red items exist, do not approve for publication
