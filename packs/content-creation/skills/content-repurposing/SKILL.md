# Content Repurposing

Use this skill when adapting existing content to a new format or channel.

Process:
1. identify the source content and its core message hierarchy
2. determine the target format and channel constraints
3. extract the essential claims, data, and narrative arc
4. adapt structure, tone, and length to the target platform
5. verify that key messages survive the transformation
6. check brand voice consistency in the new format
7. note what was dropped or simplified and why

Supported flows:
- blog post -> LinkedIn post, Twitter/X thread, email newsletter excerpt
- long-form report -> executive summary, slide deck outline
- technical article -> general-audience explainer
- text content -> video script, podcast talking points
- research findings -> stakeholder brief, infographic outline
