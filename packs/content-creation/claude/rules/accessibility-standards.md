# Accessibility Standards

All published content must meet baseline accessibility requirements:

- images: descriptive alt text that conveys meaning, not just "image of..."
- reading level: aim for grade 8-10 unless the audience requires technical depth
- link text: descriptive anchor text, never "click here" or bare URLs
- color contrast: when specifying colors, ensure WCAG AA minimum (4.5:1 for text)
- heading hierarchy: sequential, no skipped levels (H1 -> H2 -> H3)
- video content: include transcript or captions guidance
- abbreviations: expand on first use

Flag accessibility gaps during editorial review. Do not treat these as optional polish.
